# Security Policy 🛡️

## Supported Versions

We actively provide security updates and patches for the following versions of **SOC Toolkit**:

| Version | Supported          |
| ------- | ------------------ |
| 6.x.x   | :white_check_mark: |
| < 6.0   | :x:                |

---

## 🔒 Reporting a Vulnerability

The **SOC Toolkit** maintainers take security seriously. If you discover a security vulnerability or security flaw in this project, please follow responsible disclosure practices:

1. **Do NOT open a public GitHub issue** for security vulnerabilities.
2. Email the maintainer directly at **`frkndncr@github.com`** or contact via GitHub profile.
3. Include details of the vulnerability:
   - Description of the issue and impact
   - Step-by-step Proof of Concept (PoC) or reproduction steps
   - Affected modules or CLI parameters
4. We will acknowledge receipt of your vulnerability report within **48 hours** and provide a patch timeline.

Thank you for helping keep **SOC Toolkit** and its user community safe!
