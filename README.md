## вӯҗ Why Enterprise SOC Toolkit v3.0.0?

Standard threat intelligence tools often generate alert fatigue and lack actionable incident response steps. **SOC Toolkit v3.0.0** is built for real-world SOC operations:

- рҹӣЎпёҸ **Incident Response Playbook Generator**: Step-by-step containment (`iptables` / endpoint isolation), eradication, and recovery guidelines for every alert.
- рҹҹў **False Positive & Cloud Infrastructure Filter**: Automatically detects Google DNS, Cloudflare, Akamai, and AWS infrastructure to eliminate false alarms.
- рҹҢҗ **Interactive Cyber Web GUI (`soc web`)**: Zero-dependency dark-mode dashboard for web-based triage.
- рҹӘө **Automated Log & Forensics Triage (`soc triage file.log`)**: Scans log dumps, refangs IOCs, ranks critical threats, and outputs executive threat summaries.
- рҹ”“ **PowerShell & Defang Decoder (`soc decode`)**: Instantly decodes `powershell -enc` base64 payloads and defanged URLs (`hXXps://`).
- рҹ“ң **SIEM & NIDS Rule Generation**: Auto-generates Sigma rules, YARA rules, and Snort/Suricata drop statements.
- рҹҗҚ **Python SDK (`import soc_toolkit`)**: Easily embed into SOAR workflows (Shuffle, DFIR-IRIS, Cortex) or Jupyter Notebooks.

# Launch Interactive Cyber Web Dashboard
soc web

## рҹӣ пёҸ CLI Usage & Features

### 1. Single IOC Analysis & Playbook
```bash
soc 185.220.101.45 --playbook --osint
```

### 2. Export Formats (HTML & STIX 2.1)
```bash
# Interactive HTML Report
soc 185.220.101.45 --html report.html

# STIX 2.1 JSON Bundle
soc 185.220.101.45 --stix report.stix.json

# JSON / CSV / Markdown
soc 185.220.101.45 --json report.json --csv report.csv
```

### 3. Generate SIEM Detection Rules
```bash
# Generate Sigma Rule
soc 185.220.101.45 --sigma

# Generate YARA Rule
soc 44d88612fea8a8f36de82e1278abb02f --yara
```

### 4. Defang / Refang URLs & Obfuscated Command Decoding
```bash
# Defang URL
soc defang "https://evil.com/payload"
# Output: hXXps://evil[.]com/payload

# Refang URL
soc refang "hXXps://evil[.]com/payload"
# Output: https://evil.com/payload
```

## рҹҗҚ Python SDK Integration
